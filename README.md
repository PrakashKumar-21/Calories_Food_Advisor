This project is an AI-powered **Calorie Detection App** built using **Google Gemini Vision Models** and **Streamlit**.  
It analyzes any food image and provides:

 Food item identification  
 Calorie estimation  
 Health rating  
 Nutritional breakdown  
 AI-generated suggestions  

---

##  Tech Stack Used

 Technology | Purpose 

**Python** , Core programming 
**Streamlit** , Frontend UI 
**Google Gemini 2.5 Vision API**, Image understanding & calorie estimation 
**PIL (Pillow)** , Image handling 
**dotenv** , Secure API key management 

---

##  Features

### 1. AI-Based Food Recognition**
Upload any food image, and Gemini automatically detects all food items present.

### 2. Accurate Calorie Estimation**
Outputs calories for each item + total calories.

### 3. Health Suggestion**
AI tells whether the meal is healthy or not, with reasoning.

### 4. Clean & Gradient UI**
Custom-designed header + gradient button for a modern UI experience.

### 5. Secure API Handling**
API Key is stored safely in `.env`.
